Student Attendance Management System (SAMS)

A comprehensive web-based Student Attendance Management System designed to streamline attendance tracking for educational institutions.

📋 Overview
The Student Attendance Management System (SAMS) is a robust platform that enables educational institutions to efficiently manage student attendance, class information, teacher assignments, and academic sessions. The system provides separate dashboards for administrators and class teachers with role-based access control.

✨ Features

🔐 Authentication & Access Control

    Role-based login system (Administrator, Class Teacher)

    Secure email and password authentication

    User role selection at login

👨‍💼 Administrator Features

    Dashboard Analytics

        Student count monitoring

        Class management overview

        Attendance statistics tracking

        Teacher assignment management

    Class Management

        Create and manage classes

        Class listing with edit/delete functionality

    Teacher Management

        Register class teachers with complete profiles

        Assign teachers to specific classes

    Session & Term Management

        Academic session configuration

        Term management within sessions

👩‍🏫 Class Teacher Features

    Student Management

        View all students in assigned classes

        Student information including admission numbers

    Attendance Tracking

        Daily attendance marking with checkbox system

        Real-time date tracking

        Success confirmation for attendance submissions

    Attendance Reporting

        Filter attendance by date

        View individual student attendance records

        Session and term-based attendance filtering

🛠️ Technical Stack

    Frontend: HTML, CSS, JavaScript

    Backend: PHP

    Database: MySQL

    Security: Developed with cybersecurity principles by a cybersecurity student

🚀 Getting Started

Prerequisites

    Web browser (Chrome, Firefox, Safari, Edge)

    Internet connection

    Valid user credentials

Login Instructions

    Access the login panel

    Select your user role (Administrator/Class Teacher)

    Enter your registered email address

    Enter your password

    Click "Login" to access your dashboard
